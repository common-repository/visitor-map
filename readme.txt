== Description ==

Visitor Map shows number of online users and their location in a world map in a widget. You can see number of logged in users and not logged in users and their IP:s, countries and cities. You can configure the Visitor Map from the admin panel. Read more about the <a href="http://visitormap.se" target="_blank">Visitor Map</a>.

== Installation ==

- Upload Visitor Map to your website and activate.

- Add the widget to your sidebar or any widget area or use the shortcode [visitor_map] on your post or page.

- You're done.

== Frequently Asked Questions ==

= How do I generate a new Google Maps API Key? =

Go to settings for the plugin in your WordPress admin panel and generate a new key. Read more at:
https://web-dorado.com/wordpress-google-maps/installation-wizard-options-menu/configuring-api-key.html
